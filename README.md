# 🧊 Glass Sidebar UI

A modern glassmorphism sidebar with dropdown menus, smooth interactions, and a clean UI design.

## 🚀 Features

- ✨ Glassmorphism effect (blur + transparency)
- 📂 Dropdown menus (Create, Account, Profile)
- 🎯 Clean and simple JavaScript logic
- 📱 Responsive layout
- ⚡ Easy to integrate into any project

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, backdrop-filter)
- JavaScript (Vanilla)

## 📁 Project Structure


project/
├── index.html
├── styles.css
└── main.js


## ⚙️ Getting Started

1. Clone the repository:


git clone git@github.com:dina-muravska/glass-sidebar.git


2. Open `index.html` in your browser

## 🧠 How It Works

- Uses `querySelectorAll` to select all dropdown elements
- Loops through them with `forEach`
- Adds a click event listener to toggle each dropdown menu
